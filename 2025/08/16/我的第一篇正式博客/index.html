  • 💻 技术踩坑

🧑‍💻 为什么要写博客？

写博客对我来说有三个意义：

  1. 输出倒逼输入 —— 只有真正理解，才能讲清楚
  2. 沉淀知识体系 —— 防止学完就忘
  3. 记录成长轨迹 —— 未来回头看会很有意义

📚 最近在学习什么？

目前主要在学习：

  • Python 后端开发
  • Java 基础与面向对象
  • 前端基础（HTML / CSS / JS）
  • 算法与数据结构

之后会逐步深入框架和项目实战。
