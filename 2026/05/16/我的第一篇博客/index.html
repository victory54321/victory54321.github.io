## 前言

欢迎来到我的博客！

## 为什么开始写博客？

写博客的好处：

1. **知识沉淀**：把学到的内容记录下来，方便日后回顾

2. **分享交流**：结识志同道合的朋友

3. **提升自己**：写作能帮助加深理解

## 博客搭建历程

从零开始，一步步搭建了这个博客：

- 使用 Hexo 框架

- 选用 Butterfly 主题

- 部署在 GitHub Pages

- 配置了评论系统 Giscus

- 支持暗色模式和本地搜索
